# Push Admin Web 프로젝트 

## 기술 요소
- nodejs
- express
- ejs
- maria db
- bootstrap, edminlte

## 프로젝트 셋팅
1. 프로젝트 기본 골격 셋팅
```
express --view ejs 
npm install
npm audit fix --force
```

2. nodemon 설치 및 인스턴스 구동 확인
```
npm install -g nodemon
"devstart": "nodemon ./bin/www" 패키지 추가
npm run devstart
```
